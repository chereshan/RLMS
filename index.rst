Добро пожаловать в документацию библиотеки **RLMS_HSE**!
==========================================================
Данный модуль предназначен для упрощения работы пользователя с базой данных лонгитюдных обследований домашних хозяйств **RLMS-HSE**. 


Для чего создан этот модуль: 
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Для снижения входных барьеров к работе на Python с базой данных **RLMS-HSE** для тех, кто не проходил специального тому обучения. В идеале должна быть достигнута полная казуальность, при которой пользователю стоит лишь знать основные команды модуля, чтобы начать с ним работу.
